# REST-API

## Обертка для вызова REST API.

Реализовано через систему Plugin'ов.

    Плагины находятся в каталоге .\api.
    На данный момент есть два плагина для работы и 4 тестовых:
        1) api\dns_1cloud   - DNS API 1Cloud
        2) api\dns_selectel - DNS API Selectel
        3) api\test_ds_file.ps1 - тест для файла типа ps1
        4) api\test_file.psm1   - тест для файла типа psm1
        5) api\dns_selectel1.psm1 - еще один тест для файла типа psm1
        6) api\test_mod     - еще один тест для модуля PS
